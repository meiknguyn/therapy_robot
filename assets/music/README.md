# Music Files Directory

Place your ambient calming music files here.

## Supported Formats
- `.wav` (recommended)
- `.mp3`
- `.ogg`
- `.flac`

## Example Files
You can add music files like:
- `calm1.wav`
- `ocean.wav`
- `forest.wav`
- `meditation.wav`

The ambient music module will automatically detect and play these files when you use commands like:
- "play music"
- "calm me"
- "start ambient"
- "relax"
