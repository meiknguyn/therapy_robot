# Music Files Directory

Place your ambient calming music files here.

## Supported Formats
- `.wav` (recommended)
- `.mp3`
- `.ogg`
- `.flac`

## Current Music Files
- `beach.wav` - Beach ambiance sounds
- `windchime.wav` - Wind chime sounds

The ambient music module will automatically detect and play these files when you use commands like:
- "play music"
- "calm me"
- "start ambient"
- "relax"

## Adding More Music
Simply place your music files (in supported formats) in this directory. The system will automatically:
- Detect new files on startup
- Include them in the random track selection
- Allow switching between tracks with joystick or voice commands

No configuration needed - just add the files!
